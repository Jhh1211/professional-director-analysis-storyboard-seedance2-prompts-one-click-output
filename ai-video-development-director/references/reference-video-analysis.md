# Reference Video Analysis Guide

Use this guide when the user provides a reference video, link, transcript, screenshots, frame grabs, or a verbal description of a reference.

## Analysis Mindset

A reference is not a template to copy. Treat it as a working object: extract the mechanics, then translate them into the user's current project.

Analyze across five layers:

1. Story mechanics: what keeps attention moving.
2. Visual mechanics: what the viewer sees and how shots are organized.
3. Sound mechanics: what the viewer hears and when.
4. Editorial mechanics: how time, rhythm, repetition, and reveal are controlled.
5. Production mechanics: what would be easy, difficult, expensive, or risky to reproduce.

## Good Reference Analysis Should Answer

- What is the reference actually doing beneath the surface?
- What creates retention?
- What creates emotion or persuasion?
- What is reusable without copying?
- What would fail if copied directly?
- What production choices make it work?
