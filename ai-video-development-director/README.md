# AI Video Development Director

A general-purpose Codex Skill for professional video concept development.

It supports:

- rough idea diagnosis,
- reference video breakdown,
- imitation/adaptation strategy,
- concept treatment,
- script overview,
- full script,
- professional shot breakdown,
- production handoff summary.

It intentionally does not generate final AI image prompts, final AI video prompts, character asset packs, or environment prompt packs. Use downstream production skills for those.

## Install

Place this folder here:

```text
%USERPROFILE%\.agents\skills\ai-video-development-director
```

or inside a project:

```text
.agents/skills/ai-video-development-director
```

## Invoke

```text
$ai-video-development-director
I have a video idea: ...
```

## Suggested Use

1. Give a rough idea or reference video.
2. Let the skill diagnose the creative direction.
3. Confirm or adjust the direction.
4. Move to treatment, script overview, full script, then shot breakdown.
5. Hand off to specialized production or asset skills when needed.
